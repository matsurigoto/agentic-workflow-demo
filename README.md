# TaskFlow API

Task Management System API

## Setup

```bash
mvn spring-boot:run
```

## API Endpoints

- GET /api/tasks
- POST /api/tasks
- GET /api/tasks/{id}
- PUT /api/tasks/{id}
- DELETE /api/tasks/{id}
- GET /api/users
- POST /api/users/register
- POST /api/users/login

## Tech Stack

- Java 11
- Spring Boot 2.7
- H2 Database (dev)
- MySQL (prod)
